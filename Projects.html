Journify
Journify is a web application for registration and authentication of users with the use of Angular Typescript and Firebase. The users can create articles that contain a title and a text body, which can be private and publicly available for sharing with other users with an automatically generated link.
Unlimited Wordle
Wordle is a guessing game for words that have a length of 5 letters. After putting in all 5 letters the input fields fill themselves with an appropriate color depending if the position of the letter is in the right place and if the letter is present in the word that is being guessed.
